Ubuntu 20.04 配置静态网络

2020-10-19

Ubuntu 20.04 的网络配置文件位于 /etc/netplan/00-installer-config.yaml:

# This is the network config written by 'subiquity'
network:
  ethernets:
    ens33:
      dhcp4: false
      addresses: [192.168.50.30/24]
      gateway4: 192.168.50.1
      nameservers:
        addresses: [192.168.50.1, 8.8.8.8]
